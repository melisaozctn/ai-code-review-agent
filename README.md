# AI Code Review Agent

## Project Description

AI Code Review Agent is a Python-based intelligent software system that analyses Python source code files and generates structured feedback about code quality.

The system uses multiple external tools to perform:
- syntax analysis,
- style checking,
- complexity analysis,
- maintainability evaluation.

The project demonstrates an AI-assisted workflow where an intelligent agent coordinates multiple analysis tools and combines their outputs into a final report.

---

## Features

- Python syntax validation
- Code style analysis
- Cyclomatic complexity analysis
- Structured code review reports
- Modular architecture
- Testing support

---

## Technologies Used

- Python
- Pylint
- Radon
- Pytest
